## Main features

* [x] Read and highlight text and conde files

## Installation

```bash
pip3 install --upgrade pyccat
```

## Getting stats

***Read a file***
```bash
ccat /tmp/teste.txt
```

***Read a file without table***
```bash
ccat -nt /tmp/teste.txt
```

***Just highlight the file***
```bash
ccat -nt /tmp/teste.txt
```
